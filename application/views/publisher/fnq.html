  • How Do We Count Clicks

    Short URL: http://goo.gl/fCCFSr
    Long URL : http://www.filmycity.in/aged-man-character-played-by-bollywood-hero/
    Invalid Click:
    If a user clicks on http://goo.gl/fCCFSr and decides to close the link before itlink redirects to http://www.filmycity.in/aged-man-character-played-by-bollywood-hero/
    Verified Click :
    If a user clicks on http://goo.gl/fCCFSr and the link redirects to http://www.filmycity.in/aged-man-character-played-by-bollywood-hero/

  • Are there any tagline rules

    • We discourage the use of misleading tagline, because the “Bounce Rate” for such traffic is high.
    We monitor “Bounce Rate” for each publisher and those who cross the threshold level will be disqualified.
    Bounce Rate is high when users lands on a website page and they do not find any content what they are looking for.
